    Android Gradle Kotlin-DSL

    在 Android开发中，Gradle脚本的构建对我来说都是比较头疼的问题。首先是对Gradle能力的不了解，其次就是对Gradle编程语言Groovy的不熟悉。由于目前Kotlin已经成为了Android开发的第一语言，Gradle团队也为我们提供了kotlin编写Gradle脚本的可能(Kotlin-DSL)。

    在官方 Kotlin Meets Gradle 一文中提到当我们使用Kotlin编写Gradle脚本的时候，将会体会到:

    • auto-completion and content assist

    • quick documentation

    • navigation to source

    • refactoring and more

    一、使用Kotlin编写Gradle脚本

    Groovy DSL script files use the .gradle file name extension.

    Kotlin DSL script files use the .gradle.kts file name extension.

    Android项目刚创建完并不是Kotlin DSL脚本，所以我们需要将进行手动转化。

    1.修改settings.gradle 将.gradle 扩展名改为.gradle.kts

    setting > Languages & Frameworks > Kotlin > Kotlin Scriptiong中的Reload script dependencies on file change
